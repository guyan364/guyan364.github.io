快速沃尔什变换

发表于 2020-03-29 更新于 2020-03-30 分类于 学习笔记

什么是快速沃尔什变换

维基: 沃尔什转换（Walsh Transform）是在频谱分析上作为离散傅立叶变换的替代方案的一种方法。

当然 OI 中用不到那么高深

类比于 \(FFT\) , 也是对原数组进行变换 , 从而满足某种性质

有什么用

在 OI 中, 常用于与 , 或 , 异或卷积

可以在 \(O(n\log n)\) 的复杂度下完成变换与逆变换

FWT

FWT 的基本思想就是构造出变换方法

那么先来考虑与 \((\&)\) , 操作的构造

想要求 \(C_k = \sum_{i\&j=k} A_i \cdot B_j\) , 考虑构造

\[FWT( C ) = FWT( A ) \cdot FWT( B )\]

先将区间从中间分开 , 可以发现 , 前一半的最高位是 \(0\) , 后一半的最高位是 \(1\) ,

那么由于与运算的性质 , 前一半无法影响后一半 , 后一半在与 \(0\) , 时影响前一半

于是有

\[FWT( A ) = merge\left( FWT( A_1 ) + FWT( A_0 ) , FWT( A_1 ) \right)\]

其中 \(merge\) , 类似与字符串合并 , 将后一个数组拼接在前一个之后

现在得到了 \(FWT( C ) = FWT( A) \cdot FWT( B )\) , 怎么得到 \(C\) 呢?

考虑进行反演

\[UFWT( A' ) = merge\left(UFWT( {A'}_0 ) - UFWT( {A'}_1 ) , UFWT( {A'}_1 ) \right)\]

类似于与运算 , 考虑或运算

前一半可以对后一半产生影响 , 后一半不能对前一半产生影响 , 于是有

\[FWT( A ) = merge\left( FWT( A_0 ) , FWT( A_0 ) + FWT( A_1 ) \right)\]

\[UFWT( A' ) = merge\left(UFWT( {A'}_0 ) , UFWT( {A'}_1 ) -UFWT( {A'}_0 ) \right)\]

最后考虑异或运算

\[C = A \cdot B\]

\[C_0 = A_0 \cdot B_0 + A_1 \cdot B_1\]

\[C_1 = A_1 \cdot B_0 + A_0 \cdot B_1\]

设

\[X_0 = (A_0 + A_1) \cdot (B_0 + B_1) \]

\[X_1 = ( A_0 - A_1 ) \cdot ( B_0 - B_1 )\]

得

\[C_0 = \frac{ X_0 + X_1 }{2}\]

\[C_1 = \frac{X_0-X_1}{2}\]

于是就有

\[FWT( A ) = merge\left(FWT( A_0 ) + FWT( A_1 ) , FWT( A_0 ) - FWT( A_1 )\right)\]

\[UFWT( A' ) = merge\left( \frac{UFWT( {A'}_0 ) + UFWT( {A'}_1 )}{2} , \frac{UFWT( {A'}_0 ) - UFWT( {A'}_1 )}{2}\right)\]
