[BZOJ3884] 上帝与集合的正确用法

发表于 2020-03-28 更新于 2020-03-30 分类于 题解

欧拉函数

Description

给出 \(p\)

求这个:

多组询问

Solution

设 \(f(p)\) 为 \(2^{2^{2^{\cdots}}} mod \, p\) 的值

根据欧拉定理 \[a^k mod \, p = a^{k \, mod \, \phi(p) + \phi(p)}\]

则有 \[f(p)= 2^{f(\phi(p))+\phi(p)}\]

递归处理 \(f(p)\) 即可
