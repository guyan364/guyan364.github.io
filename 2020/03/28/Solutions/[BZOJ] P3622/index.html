[BZOJ3622] 已经没有什么好害怕的了

发表于 2020-03-28 更新于 2020-03-30 分类于 题解

"因为我已经不再是孤单一人了"

Description

Input

输入的数字保证全不相同

Output

Solution

\(k = ( n + k ) / 2\) 将比 \(a_i < b_j\) 多 \(k\) , 转化成 \(a_i > b_j\) 有 \(k\) 对

恰有 \(k\) 对限制太强 , 求出大于等于 \(k\) 对的方案数

可以由 dp 求出

将 \(a_n\) 排序后 , 求出 \(S_i = \sum_{i=1}^n [A_i > B_j]\)

设 \(f[i][j]\) 表示前 \(i\) 对中 , 有 \(\geq j\) 对 \(A_i > B_i\)

转移 \(f[i][j] = f[i - 1][j] + f[i - 1][j - 1] * ( S[i] - j + 1 )\)

就是分选 \(A_i\) 与不选 \(A_i\) 两种情况讨论

由于选定了 \(j\) 组 , 剩下 \(i - j\) 组任意组合 , 最后要乘上 \(( i - j)!\)

设 \(f(k)\) 表示恰有 \(k\) 组 , \(g(k)\) 表示有 \(\geq k\) 组

\[g(k) = \sum_{i=k}^n \binom{i}{k}f(i)\]

反演

\[f(k) = \sum_{i=k}^n (-1)^{(i-k)} \binom{i}{k}g(i)\]

那么答案就是 \[ans = \sum_{i=k}^n (-1)^{(i-k)}\binom{i}{k} f[n][i] \cdot (n - i)!\]
